# Login Page

A modern, interactive login page built with React, TypeScript, Vite, Tailwind CSS, and Framer Motion. 
Features interactive cartoon characters that follow your cursor and react to your inputs.

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
